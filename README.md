# FinTablo — Диагностика роста практики финансиста

## Запуск
1. Распакуй папку fintablo-quiz/
2. Открой index.html в браузере (Chrome, Firefox, Safari)
3. Для демо-режима: нажми «Запустить с демо-данными»

## Демо-профиль
Алексей: 5 клиентов, чек 30 000 ₽, 15 ч/клиент, 65% рутины, доход 150к, цель 300к

## Сброс данных
Открой консоль браузера (F12) и выполни:
```
localStorage.clear()
```
Затем обнови страницу.

## Email-отправка
Сейчас: лиды сохраняются в localStorage + открывается почтовый клиент
для отправки на maschenko@icloud.com

Для автоматической отправки без действий пользователя:
подключи Formspree (https://formspree.io) — бесплатно до 50 лидов/мес

## Просмотр сохранённых лидов
В консоли браузера:
```
JSON.parse(localStorage.getItem('fintablo_leads'))
```

- index.html — квиз (9 шагов)

- js/quiz.js — логика квиза
- js/calculator.js — расчёты метрик
